### Project Type
Academic / Sample GRC Project

### Objective
To perform a structured cybersecurity risk assessment aligned with the
NIST Cybersecurity Framework (CSF).

### Scope
This project simulates a cybersecurity risk assessment for a sample
organization by identifying assets, risks, vulnerabilities, and potential
business impact.

### Methodology
- Identified key information assets
- Analyzed threats and vulnerabilities
- Assessed risk impact and likelihood
- Prioritized risks using NIST CSF functions
- Recommended mitigation strategies

### Outcome
The assessment resulted in a prioritized risk register and high-level
mitigation recommendations to support risk reduction and security planning.

### Disclaimer
This project is for academic and learning purposes only and does not
represent a real organization or system.
